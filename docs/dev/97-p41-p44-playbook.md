# P41-P44 Playbook

## Reusable Lessons

- Start a resumed round with an unblock audit. It prevents accidental drift into environment work that still cannot be completed.
- Treat modularized page layers as needing documentation, not just code extraction. A small contributor map pays off quickly.
- On Windows, separate "preview looks broken" from "source is broken" before editing localized text.
- If an import barrel is part of the application boundary, add a contract test for it as soon as the structure stabilizes.
- When credentials are missing, invest in safety rails that make the later integration round faster and less risky.
