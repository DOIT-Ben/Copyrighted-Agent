## 实施顺序

1. 先补 provider sandbox，锁住真实 HTTP 联调路径。
2. 再补 runtime backup/restore，先以安全预演为主，不直接恢复到活跃目录。
3. 再补 metrics baseline，让真实样本结果能沉淀为固定快照。
4. 更新 `/ops` 命令入口与 runbook。
5. 做全量回归、真实工具烟测、文档收口。

- `py -m pytest` 全绿。

- `py -m app.tools.metrics_baseline` 成功生成 Markdown 与 JSON 快照。
