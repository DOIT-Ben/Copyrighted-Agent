# P45-P49 Playbook

## Reusable Lessons

- Before the secret-bearing final step, turn the workflow into one stable command and test it with a sandbox.
- Use a low-cost default smoke set for real provider validation:
  - one safe probe
  - one mode A package
  - one mode B corpus
- Keep blocked-state artifacts just as important as pass-state artifacts; they shorten the next handoff dramatically.
- A release gate should be part of the same validation workflow as probe and sample smoke, not a separate afterthought.
